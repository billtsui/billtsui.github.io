If-Then 语句

if-then 语句是所有控制流语句中最基础的一种。它指示程序仅在特定测试条件为真时执行某段代码。例如，Bicycle 类可规定刹车系统仅在自行车处于运动状态时才能降低车速。applyBrakes() 方法的一种可能实现如下： 

void applyBrakes() {
    // the "if" clause: bicycle must be moving
    if (isMoving) {
        // the "the" clause: decrease current speed 
        currentSpeed--;
    }
}
