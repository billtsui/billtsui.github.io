Java 知识体系梳理汇总

发表于 2025-11-13 更新于 2025-11-14 分类于 技术

写这些文章的初衷是什么？

中文世界的Java教材亦或者视频教程使用的JDK版本大多比较滞后，且带有传统编程思维的局限性(从面向过程到面向对象的转变)。内容编排和知识体系也比较陈旧，无法和英文世界的Java知识体系接轨。因此我决定梳理自己的Java知识体系，整理成系列文章，分享给大家。这些内容大部分来自于 dev.java，我觉得内容非常好。

主要针对哪些问题？

  1. 碎片化的Java知识体系
  2. 过时的Java编程思维
  3. 忙于业务代码，没有深度思考

都涉及哪些内容？

结合自己在Discord、Quora和Reddit等社区的交流经验，梳理出一套涵盖Java基础、进阶和高级的知识体系，内容包括但不限于：

  • Java基础语法和核心概念
  • 面向对象编程（OOP）原理
  • Java集合框架
  • 异常处理机制
  • 多线程和并发编程
  • Java I/O和NIO
  • Java 8及以上版本的新特性（如Lambda表达式、Stream API等）
  • JVM原理和性能优化
  • 常用Java开发工具和框架（如Maven、Gradle、Spring等）
  • 实践项目和代码示例

一些背景信息

  1. JDK版本: Eclipse Temurin 25 LTS for Linux x64
  2. 操作系统: Pop!_OS 22.04 LTS
  3. 编辑器: Visual Studio Code 1.105.1
