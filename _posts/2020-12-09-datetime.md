### Java 8 之前常用的日期时间类

#### 1、获取当前时间

```java.util.Date ``` 类封装了当前的日期时间。```Date```类提供两个构造函数： 



```java
Date();
Date(long millisec);
```



第一个构造函数使用当前日期和时间来初始化对象。

第二个构造函数接收一个参数，该参数是从格林尼治时间1970年1月1日00分00秒起的毫秒数。

```java
import java.util.*;

class DateTimeTest {
	public static void main(String[] args) {
		//获取系统当前日期时间
		Date date = new Date();
		System.out.println(date);
		
		//距离1970年1月1日00分00秒之后1秒的时间
		Date date1 = new Date(1000);
		System.out.println(date1);
	}
}
```

以上代码会打印两个时间

```
Wed Dec 09 11:19:53 CST 2020
Thu Jan 01 08:00:01 CST 1970
```

这里没有打印00:00:01，而是08:00:01。```Date```类的```toString()```方法会默认转换时区，因为是东八区，所以加了8个小时。如果想要得到1970年1月1日之前的时间，需要传入一个负的毫秒数。

```java
import java.util.*;
class DateTimeTest {
	public static void main(String[] args) {
		Date date = new Date(-1000);
		System.out.println(date);
	}
}
```

```
Thu Jan 01 07:59:59 CST 1970
```

#### 2、打印不同格式的时间

```SimpleDateFormat``` 类提供自定义日期时间格式，用来格式化日期。

```java
import java.util.*;
import java.text.*;

class DateTimeTest {
	public static void main(String[] args) {
		Date date = new Date();
		
		SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss:S");
		SimpleDateFormat sdf1 = new SimpleDateFormat("yyyy年MM月dd日 HH时mm分ss秒S毫秒");
		
		System.out.println(sdf.format(date));
		System.out.println(sdf1.format(date));
	}
}
```

```
2020-12-09 14:10:33:369
2020年12月09日 14时10分33秒369毫秒
```

上面的代码指定了转换格式。其中**yyyy**是完整的公元年，**MM**是月份，**dd**是日期，**HH:mm:ss**是 **时 ** **分** **秒** ，大写的**S**是毫秒。

下表是常用的日期和时间格式化编码。

| 字母 | 描述                     | 示例                    |
| :--- | :----------------------- | ----------------------- |
| G    | 纪元标记                 | AD                      |
| y    | 四位年份                 | 2001                    |
| M    | 月份                     | July or 07              |
| d    | 一个月的日期             | 10                      |
| h    | A.M./P.M. (1~12)格式小时 | 12                      |
| H    | 一天中的小时 (0~23)      | 22                      |
| m    | 分钟数                   | 30                      |
| s    | 秒数                     | 55                      |
| S    | 毫秒数                   | 234                     |
| E    | 星期几                   | Tuesday                 |
| D    | 一年中的日子             | 360                     |
| F    | 一个月中第几周的周几     | 2 (second Wed. in July) |
| w    | 一年中第几周             | 40                      |
| W    | 一个月中第几周           | 1                       |
| a    | A.M./P.M. 标记           | PM                      |
| k    | 一天中的小时(1~24)       | 24                      |
| K    | A.M./P.M. (0~11)格式小时 | 10                      |
| z    | 时区                     | Eastern Standard Time   |
| '    | 文字定界符               | Delimiter               |
| "    | 单引号                   | `                       |

#### 3、创建指定的日期时间

```Date ``` 类有几个构造函数可以根据入参构造指定的日期时间，但都被标记为过时了，这里不建议使用。可以使用 ```Calendar ``` 类创建指定的日期时间。



```java
import java.util.*;

class DateTimeTest {
	public static void main(String[] args) {
		Calendar calendar = Calendar.getInstance();
		calendar.set(2020, 11, 9,0,0,0);
		System.out.println(calendar.getTime());
	}
}
```

```
Wed Dec 09 00:00:00 CST 2020
```

```Calendar``` 类提供了一组 ```set``` 方法设置时间。需要特别注意的是，```month```参数是从0 ~ 11，而不是日常生活中的1 ~ 12。

```Calendar``` 类定义了一组常量，来表示不同的意义。同时有很多方法需要用到这些常量，具体见下表。

| 常量                  | 描述                           |
| :-------------------- | :----------------------------- |
| Calendar.YEAR         | 年份                           |
| Calendar.MONTH        | 月份                           |
| Calendar.DATE         | 日期                           |
| Calendar.DAY_OF_MONTH | 日期，和上面的字段意义完全相同 |
| Calendar.HOUR         | 12小时制的小时                 |
| Calendar.HOUR_OF_DAY  | 24小时制的小时                 |
| Calendar.MINUTE       | 分钟                           |
| Calendar.SECOND       | 秒                             |
| Calendar.DAY_OF_WEEK  | 星期几                         |

比如，我想在当前日期上往后加10天，就可以这么使用：

```java
Calendar calendar = Calendar.getInstance();
calendar.add(Calendar.DATE, 10);
System.out.println(calendar.getTime());
```

```
Sun Dec 20 13:45:11 CST 2020
```

根据add的传入的枚举类型，来分别设置字段，同时我们也可以获得一些信息。

```java
Calendar c1 = Calendar.getInstance();
// 获得年份
int year = c1.get(Calendar.YEAR);
// 获得月份
int month = c1.get(Calendar.MONTH) + 1;
// 获得日期
int date = c1.get(Calendar.DATE);
// 获得小时
int hour = c1.get(Calendar.HOUR_OF_DAY);
// 获得分钟
int minute = c1.get(Calendar.MINUTE);
// 获得秒
int second = c1.get(Calendar.SECOND);
// 获得星期几（注意（这个与Date类是不同的）：1代表星期日、2代表星期1、3代表星期二，以此类推）
int day = c1.get(Calendar.DAY_OF_WEEK);
```

### Java 8新增的日期时间类

```Date``` 和 ```Calendar``` 类一直被开发者吐槽，在参考了著名的joda-time后，Java 8新增了一组用于处理日期时间的类型。

#### 1、获取日期

```java.time.LocalDate``` 用于表示当前日期。它只有日期，不包含时间。

```java
import java.time.*;

class DateTimeTest {
	public static void main(String[] args) {
		LocalDate date = LocalDate.now();
		System.out.println(date);
	}
}
```

```
2020-12-10
```

当然也可以指定日期来构造对象。这里的month就是从1 ~ 12。而不是```Calendar``` 类的0 ~ 11。

```java
LocalDate date = LocalDate.of(2020, 1, 1);
```

```
2020-01-01
```

获取年月日信息

```java
LocalDate date = LocalDate.now();
System.out.println(String.format("%d年%d月%d日",date.getYear(),date.getMonthValue(),date.getDayOfMonth()));
```

```
2020年12月10日
```

#### 2、比较两个日期是否相等

```java
LocalDate date = LocalDate.now();
LocalDate date1 = LocalDate.of(2020, 1, 10);
System.out.println(date.equals(date1));
```

输出 ```false``` 。

#### 3、将当前日期往后设置1周。

```java
LocalDate date = LocalDate.now();
LocalDate afterDate = date.plusDays(7);
System.out.println(afterDate);
```

```
2020-12-17
```

#### 4、获取时间

使用 ```java.time.LocalTime``` 类可以获取时间，仅仅是时间，不包含日期。

```java
LocalTime now  = LocalTime.now();
System.out.println(now);
```

```20:52:17.258569
20:52:17.258569
```

#### 5 、将当前时间往后设置1小时

```java
LocalTime now  = LocalTime.now();
LocalTime afterOneHour = now.plusHours(1);
System.out.println(afterOneHour);
```

```
21:01:53.524221
```

我们也可以将时间往前设置，比如往前设置1年和1小时

```java
LocalTime beforeOneHour  = LocalTime.now().minusHours(1);
System.out.println(beforeOneHour);
		
LocalDate beforeOneYear = LocalDate.now().minusYears(1);
System.out.println(beforeOneYear);
```

```
20:15:03.792915
2019-12-10
```

```LocalDate``` 中还提供了 ```isAfter()``` ,```isBefore()```,```isLeapYear()``` 这样快速便捷的方法，来判断日期的先后关系，当前年份是否为闰年。 

```java
LocalDate date = LocalDate.now();
LocalDate date1 = LocalDate.now().minusYears(1);
System.out.println(date1.isBefore(date));
		
System.out.println(date.isLeapYear()?date.getYear()+"年是闰年":date.getYear()+"年不是闰年");
```

```
true
2020年是闰年
```

上面两个类是分别针对日期和时间的，可以使用 ```LocalDateTime``` 类获取完整的日期时间对象。 

```java
LocalDateTime now = LocalDateTime.now();
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss");
System.out.println(now.format(formatter));
```

```
2020-12-10 21:49:17
```

#### 6、时间戳

```Instant``` 类用于时间戳运算。它以1970年1月1日0时0分0秒0000毫秒为起始，计算当前时间距离起始时间经过了多少毫秒。

```java
Instant i = Instant.now();
System.out.println(i);
System.out.println(i.toEpochMilli());
System.out.println(System.currentTimeMillis());
```

```
2020-12-10T13:59:15.487441Z
1607608755487
1607608755503
```

#### 7、获取时间段

有一个特别有用的类 ```Duration``` 用来表示一段时间。比如你可以用Duration对象存储两个不同的日期时间间隔，然后获取间隔的天数，小时数，秒数等等。

```java
LocalDateTime from  = LocalDateTime.of(2020, 10, 10,0,0);
LocalDateTime to = LocalDateTime.now();
Duration d = Duration.between(from, to);
System.out.println("两个日期间隔"+d.toDays()+"天");
System.out.println("两个日期间隔"+d.toHours()+"小时");
System.out.println("两个日期间隔"+d.toMinutes()+"分钟");
```

```
两个日期间隔61天
两个日期间隔1486小时
两个日期间隔89178分钟
```
